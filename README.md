# Sleep Breathing Irregularity Detection

This project detects breathing irregularities during sleep using physiological signals and deep learning. The system analyzes overnight sleep recordings and classifies breathing patterns using a **1D Convolutional Neural Network (CNN)**.

The complete methodology, implementation details, and results are described in **report.pdf**.

---

## Project Overview

The objective of this project is to identify abnormal breathing patterns such as **Hypopnea** and **Obstructive Apnea** using physiological sleep signals.

Signals used in the model:

* Nasal Airflow (32 Hz sampling rate)
* Thoracic Movement (32 Hz sampling rate)
* SpO₂ Oxygen Saturation (4 Hz sampling rate)

The dataset also contains **sleep_profile** data (sleep stage information), but this information was **not used in this assignment or model training**, since the task specifically focuses on detecting breathing irregularities from physiological breathing signals.

---

## Pipeline

The implemented pipeline consists of the following stages:

1. Signal visualization
2. Signal preprocessing and filtering
3. Window segmentation (30 seconds with 50% overlap)
4. Dataset creation and labeling
5. CNN model training
6. Leave-One-Participant-Out cross validation


## Important Note About Dataset Files

The generated dataset files:

sleep_breathing_windows.csv

exceed **100MB**, which is larger than the GitHub file size limit.

Therefore, these files were **not pushed to this repository**.

They can be regenerated by running the dataset creation script.

---

## Usage

### Generate signal visualizations

python scripts/vis.py -name "Data/AP01"

This generates a PDF visualization of the participant’s physiological signals.

---

### Create dataset

python scripts/create_dataset.py -in_dir "Data" -out_dir "Dataset"

This script:

* Filters the signals
* Creates 30-second windows
* Assigns labels based on event annotations
* Saves the dataset

---

### Train the model

python scripts/train_model.py

This script trains the CNN model and evaluates it using **Leave-One-Participant-Out cross validation**.

---

## Results

The CNN model achieved an average accuracy of approximately **91%** across participants.

Detailed evaluation results and analysis are available in **report.pdf**.
